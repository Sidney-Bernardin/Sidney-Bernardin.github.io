Project 1

Destiny2 Webhook For GoogleHome

A Destiny Item Manager (DIM) that I created with Actions on Google. A Destiny Item Manager can do things like move and equip items within the game of Destiny2.

Technologies

  • Golang: Go is what I wrote the webhook in. I made great use of golangs concurrency featurs to reduce latency.
  • Datastore: I chose Datastore for my database because its very easy to implement when using other GCP tools.
  • Google Cloud Functions: Deploying to GCF makes for very easy maintaining and scalability .
  • Google Home Actions: The action is what makes requests to the golang webhook.

Project 2

Recipe Share

This is a fullstack, mivroservice based application that allows people to post and share recipes.

Technologies

  • Go: Go is what I used for the hexagonal based backend services.
  • MongoDB: I chose MongoDB because its a fast noSQL database that suited well for my use case.
  • VueJS: I chose VueJS to do the UI simply because thats my favotite frontend framework.
  • Cloud Run: I didn't have all that many mivroservices to deploy witch made deploying to Google Cloud Run an great choice.

Architecture

I went with multiple hexagonal based mivroservices for the backend. hexagonal services make for great testability.
