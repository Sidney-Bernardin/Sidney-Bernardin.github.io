destiny 2 google home action

/Source Code

Description

My Google Home action is a Destiny Item Manager (DIM) that I created with Actions on Google. A Destiny Item Manager can do things like move and equip item within the game of Destiny.

Technologies

  • Go: Go is what I wrote the webhook in and I made great use of golangs concurrency featurs.
  • Datastore: I chose Datastore for my database because its a noSQL db and its very easy to implement when it comes to a GCF.
  • Google Cloud Functions: Deploying on GCF makes for very easy scalability.
  • Google Home Actions: The action is what makes requests to the golang webhook thats hosted on GCF.

Recipe Share

Description

This is a full stack application that allows people to post and share recipes.

Technologies

  • Go: Go is what I used for the hexagonal based backend.
  • MongoDB: I chose MongoDB because a fast noSQL database suited well for my use case.
  • VueJS I chose VueJS to do the frontend in simply because thats my favotite frontend framework.
  • Cloud Run: I deployed this application on Cloud Run but the code-base makes it very easy to doploy many ways.

Architecture

I went with a Hexagonal based backend because its testability combined with its simplicity makes for a great development experience.
